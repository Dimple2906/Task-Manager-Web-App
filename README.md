
Task Manager Web Application

A simple and professional task management web application built using HTML, CSS, and Vanilla JavaScript. The application supports user authentication and full task CRUD operations using browser LocalStorage.

Features

User signup and login

Create, view, edit, delete tasks

Task status management (pending/completed)

Dashboard with task statistics

Search and filter tasks

Light and dark mode toggle

Responsive and clean UI

Technologies Used

HTML5

CSS3

JavaScript (ES6)

LocalStorage

How to Run

Open signup.html to create an account, then log in using login.html. No server setup required.

Data Storage

All user and task data are stored locally in the browser using LocalStorage. Tasks are user-specific.

Project Use Case

This project demonstrates frontend fundamentals such as DOM manipulation, state handling, CRUD operations, and UI design best practices.
